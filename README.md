# Fashion MNIST classifier

Used multiclass classification (one-vs-rest) with a softmax function, built an API to serve the trained model, and created a React-based web application that can take an image from the local system or a camera feed for classification.

## running frontend

Need Nodejs intalled

```bash
cd frontend
npm install
npm run dev
```

## running backend

Need Python installed

```bash
cd backend
pip install -r requirements.yxt
fast api dev
```

## running and training a model

python3 -m training.train
